Detailed Description

obsługa api ollama

Klasa odpowiedzialna za komunikację z lokalnym serwerem Ollama poprzez REST API.

  • Klasa wykorzystuje bibliotekę CPR do wykonywania żądań HTTP i nlohmann/json do przetwarzania danych w formacie JSON. Umożliwia wysyłanie zapytań do modelu Bielik[cite: 7, 13, 19].
