\doxysubsection{Detailed Description}
Główna klasa interfejsu użytkownika aplikacji Redaktor AI. 

Klasa odpowiedzialna za główne okno aplikacji Redaktor AI.


\begin{DoxyItemize}
\item Klasa zarządza interfejsem graficznym, obsługuje interakcje z użytkownikiem oraz koordynuje komunikację z modelem językowym w osobnym wątku.
\item Zarządza interfejsem użytkownika, pobiera dane wejściowe i wyświetla zredagowany tekst otrzymany z modelu językowego. 
\end{DoxyItemize}
